Class and Instance

  • Class (or Type) is the name given to a generic description of an object.

  • Instance (or object) : An individual object of a certain class

  • For example,

    • "Banana" is an instance of the class "Fruit"
    • "X220" is an instance of the class "Laptop Computer"

Referencing Attribute

Use "."

  • primary.identifier
  • If the primary is an object (instance), it first checks if the object has an instance attribute of "identifier".
  • If there is no instance attribute or if the primary is a class, it checks for a class attribute of "identifier".
  • Similar to name space look up (local , global)

string

a string is a sequence of characters

  • "abc"

    • str is a class, and "abc" is an instance of str.
    • find, replace, etc
    s = "abcdefghijk"
s.find("c") # 2
s.find("gh") # 6
s.find("ghb") # -1, search failed
s1 = s.capitalize().replace("Abc","ABC")
print s1 # 'ABCdefghijk'

Lists

finite ordered sets of arbitrary Python objects.

  • created by placing a comma-separated list of expressions in square brackets.

    • [1, 2, 3]
    • [1, "abc", 3.]

  • indexed by non-negative numbers

  • The built-in function len() returns the number of items

    • len([1,2,3]) $\rightarrow$ 3

Indexing and Slicings

Use "[]"

  • Indexing (Subscriptions)

    primary[expression]

  • Slicings

    primary[start:end]
primary[start:end:stride]
    • stride is optional and default is 1

Tuples

Similar to the list, but is immutable.

  • immutable object

    • an object whose state(attributes) cannot be modified after it is created.
t = ("one", "two", "three")
print t[0] # "one"
t[1] = "twenty" # exception is raised

List vs. Tuples

Tuples have structure, lists have order.

  • Tuples (generally) are sequences of different kinds of stuff, and you deal with the tuple as a coherent unit.
  • Lists (generally) are sequences of the same kind of stuff, and you deal with the items individually.

Dictionary

for statement

for loop is most common control flow in Python.

for_stmt ::=  "for" target_list "in" expression_list ":" suite
              ["else" ":" suite]

For example,

a = ['cat', 'window', 'defenestrate']
for x in a:
   print x, len(x)

Make a copy when the list is modified during the loop

for x in a[:]: # make a slice copy of the entire list
    if len(x) > 6: a.insert(0, x)
