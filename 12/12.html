FITS

http://fits.gsfc.nasa.gov/fits_standard.html

A fits file is a list of HDU.

  • HDU : Header Unit (Header + Data)

    • Header : header that descrives the data
    • Data : actual data. can be an image or a table.

The first HDU must be an image HDU (or empty). The other HDU can be an image HDU or an table HDU.

pyfits

pyfits.open : creates an instance of HDUList.

HDU :

  • header : dict-like object
  • data : numpy array (recarray for table data)

See tutorial.

More details can be found from

http://stsdas.stsci.edu/download/docs/The_PyFITS_Handbook.pdf

WCS - World Coordinate System

A mapping between pixel coordinate and celetial coordinate.

Relavant header keyword

  • ctypeN : type of coordinate including a projection
  • crpixN : reference pixel
  • crvalN : coordinate at reference pixel
  • cdeltN : delta coordinate for a single pixel

But this is not all, there is more!

For a simple case,:

coordinate_at_pixel_x = crvalN + cdeltN * (x - crpixN)

BUT, note that in fits convention, the pixel coordinate of the upper-left corner is (1, 1) NOT (0, 0) as in numpy.

saods9

ds9 provides xpa protocol with external processes.

http://hea-www.harvard.edu/RD/ds9/ref/xpa.html

pysao is a wrapper around the xpalib and helper classes that enables you easy communication with ds9.

import pysao
ds9 = pysao.ds9()
ds9.view(np.arange(100).reshape((10, 10)))

f = pyfits.open("test.fits")
ds9.view(f[0])
